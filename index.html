Prueba concepto Xamarin.

Este aplicativo esta desarrollado para iphone, y consulta servicios web (http://1.xenon-world-234.appspot.com/) que responden json

El menu principal se obtiene presionando cualquier parte de la pantalla y deslizar hacia la derecha(--->)

Contiene 4 Items:

  • Inicio
  • Acerca de oscar
  • Proyecto moviles
  • Proyectos Web
  • Contacto

La mayoria de Item cargan un json Para los proyectos moviles se carga un webView.

Este proyecto se desarrollo en xamarin + storyBoards de iphone
